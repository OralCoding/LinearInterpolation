## 선형 보간 (Linear Interpolation)

선형 보간 (Linear Interpolation)은 두 점 사이에 있는 값들을 추정하는 과정입니다. 

두 점 (x1, y1)와 (x2, y2)를 알고 있다면, 이 두 점 사이의 임의의 x 값에 대한 y 값을 다음과 같은 공식으로 찾을 수 있습니다:

y = y1 + ((y2 - y1) * (x - x1)) / (x2 - x1)
